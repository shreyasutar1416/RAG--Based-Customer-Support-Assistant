🧠 RAG-Based Customer Support Assistant

A Retrieval-Augmented Generation (RAG) powered customer support assistant that answers user queries using a PDF knowledge base and escalates complex cases to human support.
🚀 Overview

This project implements an intelligent customer support system that:

    📄 Processes a PDF knowledge base (Customer Service Handbook)
    🔍 Retrieves relevant information using semantic search
    🤖 Generates context-aware answers using an LLM
    🔀 Routes queries based on intent
    👨‍💼 Escalates complex queries via Human-in-the-Loop (HITL)

🧩 Key Features

    ✅ Retrieval-Augmented Generation (RAG)
    ✅ PDF-based knowledge ingestion
    ✅ Semantic search using embeddings
    ✅ Vector database (ChromaDB)
    ✅ Fast LLM inference using Groq
    ✅ Intent-based routing
    ✅ Human-in-the-Loop escalation

🛠️ Tech Stack

    Language: Python
    Frameworks: LangChain
    Vector DB: ChromaDB
    LLM: Groq (LLaMA 3.1)
    Embeddings: HuggingFace (MiniLM)
